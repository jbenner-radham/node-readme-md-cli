readme-md-cli
=============
A CLI for the `readme-md` module (better description forthcoming...)

Install
-------
```sh
$ npm install --save readme-md-cli
```

Usage
-----
```sh
$ readme-md --help

  Usage
      $ readme-md

  Options
      --help, -h       Display this message.
      --version, -v    Display the application version.
```

Testing
-------
```sh
$ npm test
```
